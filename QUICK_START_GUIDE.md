# VibeCheck Quick Start Guide ⚡

Get up and running in **5 minutes**!

---

## 📱 Installation (1 minute)

```
1. Open Google Play Store
2. Search "VibeCheck"
3. Tap "Install"
4. Wait ~30 seconds
5. Tap "Open"
```

Done! ✅

---

## 🔐 Account Setup (2 minutes)

### Step 1: Welcome Screen
```
Tap "Let's Begin"
```

### Step 2: Age Verification
```
Select your age range
(Used for appropriate content filtering)
```

### Step 3: Choose Username
```
Enter a unique username (3-20 characters)
Example: "wellness_warrior" or "mood_tracker_21"
```

### Step 4: Phone Verification
```
Enter your phone number: +[country code][number]
Example: +1 234 567 8900

Tap "Send OTP"
↓
You'll receive 6-digit code via SMS
↓
Enter the code in the app
↓
Phone verified! ✅
```

### Step 5: Profile Creation
```
First Name: [Your name]
Last Name: [Your name]
Avatar: (Optional - tap to upload a photo)

Tap "Finish"
```

**You're in!** 🎉

```
Home Screen:
```

### 2. Complete Today's Gauntlet Quests
```
Tap "Gauntlet" tab
↓
See 3 quests for today
↓
Complete all 3:
├─ Morning Check-in (2 min)
├─ Mood Journal (3 min)
└─ Gratitude Practice (1 min)
↓
Get 30+ gems! 💎
```

### 3. Try the Resonance Feed
```
Tap "Feed" tab
↓
See posts from worldwide
↓
Tap 🔥 "Resonate" on posts you relate to
↓
Scroll to bottom → Tap + to share your own mood
```

## 🎯 Main Features at a Glance

### **RESONANCE FEED 🔥**
```
What:    Share mood + photo anonymously
Why:     Connect with people worldwide
How:     Tap + button → Select mood → Write message → Post
Earn:    Gemstones for engagement
Time:    2 minutes
```

### **THE GAUNTLET 🎮**
```
What:    3 daily mood quests
Why:     Build healthy habits + earn rewards
How:     Tap each quest → Answer → Get gems
Earn:    5-15 gems per quest + streak bonuses
Time:    5 minutes
```

### **FRIENDS 👥**
```
What:    Connect with real people
Why:     Support each other's mood journey
How:     Add Friend → Search by name/phone → Send request
Track:   See friends' streaks & moods
Time:    1 minute to add a friend
```

### **INSIGHTS 📈**
```
What:    See your mood patterns
Why:     Understand what affects your mood
How:     Settings → Insights → View charts
Export:  Download as CSV (Plus only)
Time:    2 minutes
```

## 🔥 Streak System Explained

```
Complete all 3 Gauntlet quests = 1-day streak 🔥

Day 1-2:   Normal gem rewards
Day 3:     1.5× multiplier activated
Day 7:     2× multiplier + 50 bonus gems + badge 🏆
Day 14:    2.5× multiplier + 100 bonus gems + badge
Day 30:    Legendary badge + 250 bonus gems 👑

Miss 1 day = Streak resets to 0 😢
```

**Pro tip:** Set daily reminder to complete quests!

## 🚫 Common Mistakes to Avoid

```
❌ NOT setting daily reminders
   → You'll forget quests
   → Streak resets
   ✅ Set reminder NOW

❌ Only doing 1-2 quests
   → Won't get streak credit
   → Miss 3-quest bonus
   ✅ Complete all 3

❌ Never checking Insights
   → Miss understanding patterns
   → Lose valuable data
   ✅ Review insights weekly

❌ Being too private with friends
   → Won't feel supported
   → Miss community benefit
   ✅ Add a few trusted friends

❌ Not trying Resonance Feed
   → Miss community aspect
   → Miss feeling connected
   ✅ Share at least 1 post
```
