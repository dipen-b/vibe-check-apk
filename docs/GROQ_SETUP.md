# Groq AI Integration Setup

This guide walks you through deploying Groq AI-powered conversation starters in VibeCheck's anonymous match feature.

## Overview

**What it does:**
- When users find a match, Groq AI generates 3 contextually-relevant opening messages
- Messages are mood-aware (based on both users' moods)
- Users can select a suggestion or type their own message
- Future: in-chat reply suggestions

**Cloud Functions:**
- `generateOpeningSuggestions(peerMood, userMood)` → returns opening message suggestions
- `generateReplySuggestions(sessionId, lastMessage)` → returns reply suggestions (future)

---

## Prerequisites

1. **Firebase project** with Cloud Functions enabled (Blaze plan)
2. **Groq API account** — free tier available at https://console.groq.com
3. **Groq API key** — generate from your Groq console dashboard
4. **Firebase CLI** — installed and authenticated

---

## Step 1: Get a Groq API Key

1. Go to https://console.groq.com/login
2. Sign up (free) or log in
3. Navigate to **API Keys** section
4. Click **Create API Key**
5. Copy the key (you'll use it in the next step)
6. **Keep this key secure** — treat it like a password

---

## Step 2: Add Groq API Key to Firebase Secrets

### Via Firebase Console (easier for beginners):

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Select your VibeCheck project
3. Navigate to **Settings** → **Service Accounts**
4. Click **Manage All Service Accounts**
5. Find your Firebase service account
6. Under **Keys**, click **Add Key** → **Create New Key** → **JSON**
7. (Skip if you already have a key)

### Via Firebase CLI (recommended):

```bash
# Ensure you're in the backend/functions directory
cd backend/functions

# Create a secret named GROQ_API_KEY
firebase functions:secrets:set GROQ_API_KEY

# When prompted, paste your Groq API key (from Step 1)
# Your key will be encrypted and stored in Google Secret Manager
```

Verify the secret was created:
```bash
firebase functions:secrets:list
```

You should see `GROQ_API_KEY` in the list.

---

## Step 3: Deploy Cloud Functions

```bash
# From backend/functions directory
npm install

# Deploy all functions (including groq.js)
firebase deploy --only functions

# Or deploy just Groq functions:
firebase deploy --only functions:generateOpeningSuggestions,functions:generateReplySuggestions
```

Expected output:
```
✔  Deploy complete!

Function URL: https://us-central1-your-project.cloudfunctions.net/generateOpeningSuggestions
Function URL: https://us-central1-your-project.cloudfunctions.net/generateReplySuggestions
```

## Step 5: Verify on Device

Once deployed, the Android app will:

1. **Detect a match** → `requestMatch()` succeeds
2. **Fetch suggestions** → `ChatRepository.getOpeningSuggestions()` calls Cloud Function
3. **Show UI** → User sees 3 suggestion cards in `MatchedWithSuggestionsContent`
4. **User picks one** → Suggestion is pre-filled in the chat input

### Test Flow:

1. Build & install the app
2. Navigate to Match tab
3. Enable anonymous chat
4. Tap "Find someone"
5. When matched, you should see **3 opening suggestions** (generated by Groq)
6. Select one and start chatting

If suggestions don't appear:
- Check Firebase Cloud Function logs: `firebase functions:log`
- Verify `GROQ_API_KEY` secret exists: `firebase functions:secrets:list`
- Check app logcat for errors: `adb logcat | grep vibecheck`

## Troubleshooting

### "Cloud Function returned null suggestions"
**Cause:** Groq API key not set or invalid
**Fix:**
```bash
firebase functions:secrets:set GROQ_API_KEY
# Re-enter your Groq API key from console.groq.com
firebase deploy --only functions
```

### "Function returned error: unauthenticated"
**Cause:** User is not signed in to Firebase
**Fix:** Ensure anonymous authentication is enabled in Firebase console

### "Timeout waiting for Groq API"
**Cause:** Groq API slow or unreachable
**Fix:** Fallback suggestions will be used. Check Groq status at https://status.groq.com

### "Invalid JSON in response"
**Cause:** Groq model returned unparseable format
**Fix:** Code has fallback parser (line-split). Suggestions will still be generated.

## Customizing Suggestions

To change the tone, style, or constraints of suggestions:

Edit `backend/functions/groq.js`, lines 90-94 (opening) or 165-170 (reply):

```javascript
const systemPrompt = `You are a friendly, empathetic conversation starter...
// Modify this prompt to change the style, tone, or constraints
`;
```

Re-deploy:
```bash
firebase deploy --only functions:generateOpeningSuggestions
```

## Security Notes

- ✅ API key stored in **Google Secret Manager** (encrypted)
- ✅ Functions are **authenticated** (Firebase Auth required)
- ✅ Rate limiting: 30 req/min on free tier (configurable)
- ✅ No suggestions are saved — ephemeral only
- ✅ Chat content is NOT sent to Groq (only moods + last message)
